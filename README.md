# Word Transformations

An interactive web app for English learners that teaches how words change form across grammatical categories — noun → verb → adjective → adverb and beyond.

**Live demo:** https://beizaaa.github.io/word_transformation/

---

## Features

**Explorer** — search any word and see its full word family as a visual graph  
**Categories** — browse all nouns, verbs, adjectives, adverbs, prepositions, and more  
**Quiz** — identify a word's category or find the correct transformation; tracks score and streak  
**Rules panel** — 14 suffix/prefix transformation rules with live examples

---

## Dataset

125 word families including: *help, work, play, create, learn, develop, think, care, hope, change, strong, happy, free, real, beauty* and more — each with definitions, example sentences, and all derived forms.

---

## Transformation rules

`-tion` `-ness` `-ly` `-ful` `-er/-or` `-less` `-ment` `un-` `re-` `-able` `-ize/-ify` `-ous` `-ive` `-ity`

---

## Tech

Pure HTML + CSS + Vanilla JavaScript. No frameworks, no dependencies, no build step.
